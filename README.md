# sql-createdb
Проект БД для компании по грузоперевозкам «Ядер-Колы» в постъядерной Америке, спроектированный с соблюдением НФ и НФБК. 

Компания перевозит груз (Ядер-Кола и сопутствующие товары),
между поселениями,
с помощью транспорта,
под управлением водителей,
по маршрутам,
с учетом заказов, рейсов, статусов и повреждений.
| Сущность    | Назначение                        |
| ----------- | --------------------------------- |
| settlements | Поселения (города, убежища, базы) |
| clients     | Клиенты (фракции, бары, убежища)  |
| cargo_types | Типы груза                        |
| vehicles    | Транспорт                         |
| drivers     | Водители                          |
| routes      | Маршруты                          |
| orders      | Заказы клиентов                   |
| order_items | Состав заказа                     |
| trips       | Фактические рейсы                 |
| trip_events | События рейса (атаки, поломки)    |
